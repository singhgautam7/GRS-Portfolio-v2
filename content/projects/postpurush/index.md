A modern, local-first API client for testing, analyzing, and documenting APIs — running entirely in the browser. Features include collections with environment variable support, an analytics dashboard, an API documentation generator with themed PDF export, and a full testing suite covering assertions, load testing, rate limit detection, and uptime monitoring. All data is stored in IndexedDB with no backend required.
